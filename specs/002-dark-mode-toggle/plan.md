# 實作計畫：深色模式切換

**分支**：`002-dark-mode-toggle` | **日期**：2026-01-29 | **規格**：[spec.md](./spec.md)
**輸入**：來自 `/specs/002-dark-mode-toggle/spec.md` 的功能規格

## 摘要

為待辦事項應用程式新增深色模式切換功能。使用者可透過標題區域的切換按鈕在淺色與深色主題之間即時切換。系統會自動偵測使用者的系統偏好設定（`prefers-color-scheme`）作為首次使用的預設值，並將使用者選擇保存於 localStorage 以便下次載入時自動套用。

技術方法：利用現有的 CSS 變數架構，透過切換 `<html>` 元素的 `data-theme` 屬性來覆寫顏色變數，實現主題切換。

## 技術情境

**語言/版本**：HTML5、CSS3、JavaScript ES6（原生）  
**主要相依套件**：無外部相依（純前端原生實作）  
**儲存方式**：localStorage（已存在於專案中）  
**測試工具**：手動測試（專案無自動化測試框架）  
**目標平台**：現代瀏覽器（支援 CSS 變數、localStorage、prefers-color-scheme）  
**專案類型**：單一網頁應用程式  
**效能目標**：主題切換在 0.3 秒內完成（SC-001）  
**約束條件**：觸控目標至少 44x44 像素（SC-004）、WCAG AA 對比度標準（SC-002）  
**規模/範圍**：單一頁面、約 3 個檔案需修改
