# 快速開始指南：使用者登入機制

**功能分支**：`003-user-login`
**建立日期**：2026-02-09

## 先決條件

- Python 3.x（用於靜態檔案伺服器）
- 現代瀏覽器（Chrome、Firefox、Safari、Edge）
- 已完成 001-todo-webapp 和 002-dark-mode-toggle 的功能

## 開發環境啟動

```bash
# 1. 切換到功能分支
git checkout 003-user-login

# 2. 啟動開發伺服器
cd src
python server.py

# 3. 開啟瀏覽器
# 訪問 http://localhost:8000
```

## 功能驗證步驟

### 登入流程

1. 開啟 `http://localhost:8000`
2. 確認看到登入畫面（帳號欄位、密碼欄位、登入按鈕）
3. 確認游標自動對焦在帳號欄位
4. 輸入帳號 `admin`、密碼 `admin`
5. 點擊「登入」按鈕
6. 確認看到短暫的「歡迎回來」訊息
7. 確認 1-2 秒後進入待辦事項主畫面

### 登入失敗

1. 在登入畫面輸入錯誤帳號（例如 `user`）和密碼 `admin`
2. 確認顯示「帳號或密碼錯誤」
3. 確認密碼欄位已清空，帳號欄位保留

### 空白去除

1. 在帳號欄位輸入 `admin`（前後有空白）
2. 在密碼欄位輸入 `admin`
3. 確認可以成功登入

### 工作階段保留

1. 成功登入後
2. 按 F5 重新整理頁面
3. 確認仍在主畫面（不需重新登入）

### 登出

1. 確認 header 右上角有登出按鈕（在深色模式切換按鈕旁）
2. 點擊登出按鈕
3. 確認回到登入畫面

### 深色模式

1. 先登入後切換到深色模式
2. 登出
3. 確認登入畫面也呈現深色模式

### 響應式設計

1. 使用瀏覽器開發者工具切換至手機視窗大小（320px 寬）
2. 確認登入畫面元素正確排列，可正常使用
3. 切換至 1920px 寬度，確認顯示正常
